# Python GUI Application

Простое каркасное приложение на Python с графическим интерфейсом пользователя (GUI), созданное на основе PyQt5.

## Структура проекта

```
PythonApp/
├── main.py              # Точка входа приложения
├── requirements.txt     # Зависимости проекта
├── README.md           # Этот файл
├── .gitignore          # Файлы, игнорируемые Git
└── app/                # Основной пакет приложения
    ├── __init__.py     # Инициализация пакета
    └── window.py       # Главное окно приложения
```

## Требования

- Python 3.7 и выше
- PyQt5

## Установка

1. Клонируйте репозиторий или скачайте файлы проекта

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск приложения

Запустите приложение, выполнив:

```bash
python main.py
```

Или с явной поддержкой Python 3:

```bash
python3 main.py
```

## Функциональность

- **Поле ввода**: Введите текст и отправьте его
- **Кнопка "Отправить"**: Обрабатывает введенный текст
- **Кнопка "Очистить"**: Очищает все поля
- **Менюбар**: Содержит меню "Файл" и "Справка"
- **Область вывода**: Отображает результаты работы приложения

## Как расширить приложение

1. **Добавить новые компоненты**: Отредактируйте `app/window.py` и добавьте новые виджеты в метод `init_ui()`

2. **Создать новые модули**: Добавьте новые файлы в папку `app/` для логики приложения

3. **Добавить обработчики**: Создайте новые методы в классе `MainWindow` для обработки событий

## Примеры расширений

### Использование разных фреймворков GUI

По умолчанию используется **PyQt5**. Вы также можете использовать:

- **Tkinter** (встроенный в Python, не требует установки):
  ```python
  import tkinter as tk
  ```

- **PySimpleGUI** (более простой в использовании):
  ```bash
  pip install PySimpleGUI
  ```

- **Kivy** (для кросс-платформенных приложений):
  ```bash
  pip install kivy
  ```
